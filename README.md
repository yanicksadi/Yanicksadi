## 🌟 Featured Project

### 💼 [**TrackItFinance App**](https://github.com/Yanick-sadi/TrackItFinance-App)

A web-based finance management system designed for both organizations and individuals.

### 🔑 Key Features

* 💰 **Employers:**

  * Manage loan applications
  * Approve and track loans
  * Monitor organizational finances through dashboards

* 💱 **Employees:**

  * Track savings
  * View loan applications and status
  * Access financial summaries

### ⚙️ Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express
* **Database:** MySQL

🧩 *Goal:* Build a transparent, efficient, and automated systems.
